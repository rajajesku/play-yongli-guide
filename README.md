# play-yongli-guide

本仓库用于归档和发布多个独立的 HTML 页面。这些页面不面向特定网站或域名，仅作为通用网页资源存放与展示的场所。

## 项目简介

`play-yongli-guide` 是一个以 HTML 页面为主的内容归档仓库。你可以在这里找到一系列功能独立、结构完整的网页文件。每个页面都是自包含的，可以直接在浏览器中打开使用，无需额外依赖。

## 目录说明

- `/` — 根目录，存放主要的 HTML 页面文件。
- 每个 HTML 文件代表一个独立的页面，文件名通常反映其用途或主题。
- 暂未划分子目录，所有页面文件均位于根目录下，便于直接访问。

## 页面归档说明

- 所有页面均为纯前端实现，不涉及后端服务。
- 页面内容可能包含静态文本、样式、脚本或简单的交互功能。
- 归档时尽量保持原始结构和功能完整性。
- 不保证所有页面在所有浏览器或设备上完全兼容。

## 维护说明

- 本仓库不定期更新，新增或修正页面内容。
- 欢迎提交 Issue 或 Pull Request 来改进页面质量或修复问题。
- 不接受涉及具体商业推广、特定平台绑定或诱导分享的内容提交。
- 维护者保留对页面内容进行修改或移除的权利。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开任意 HTML 文件即可查看。
3. 也可通过 GitHub Pages 等服务部署后在线访问（需自行配置）。
